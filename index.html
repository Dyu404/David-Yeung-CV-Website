Hi. I'm David

I am trained as an artworker/graphic designer, but I made the brave choice of switching careers during the 2020 pandemic. With people working, studying and meeting up from thier homes has made me think how exciting and marvelous web technology can be. It's like the great space race in the 1960's, which saw two great nations compete to put the first man in space, and eventually onto the moon. The techology which came to the world from these events has benifited humankind so much, so I believe web technologies and the way we see the web will change, and to be part of it will be like part of something special.

I've always loved technology, so learning how to code is something I see as a hobby rather than a chore. I believe I have come a long way since starting the first HTML and CSS online course on Udemy in January 2021, but it's quite exctiing to see how many other new things I can still learn. I plan to extend into Javascript, Materialize and React once I have a proficient knowledge of HTML5 and CSS. As side projects, I plan to create websites for friends and family, and also delve into making javascript games.
